Addendum

Server Setup

I set up an Express.js server to handle both static file serving and form submission. Middleware such as express.urlencoded was used to parse URL-encoded form data, and morgan was used for logging HTTP requests.

Form Design

The form was designed to be simple yet functional, with fields for noun, verb, adjective, adverb, and place. All fields are marked as required to ensure complete user input. Labels are provided for clarity.

Routing and Handling

A POST route /ITC505/lab-7/index.html was implemented to handle form submissions. The route validates the input, ensuring no fields are left empty, and constructs a mad lib story using the submitted data. If validation fails, a response prompts the user to fill out all fields.

CSS Styling

The form and page were styled using CSS to improve usability and aesthetics.

Verification

The page was tested to ensure that it loads correctly, form validation works, and the mad lib story is generated as expected. The last modified date is displayed correctly in the footer.
