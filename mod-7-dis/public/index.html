HTTP Status Codes

  • 418 I'm a Teapot

    Numerical Code: 418

    Purpose: This code was defined in 1998 as one of the traditional IETF April Fools' jokes, in RFC 2324, Hyper Text Coffee Pot Control Protocol.

    Situation: A coffee pot should return this error code if an attempt is made to brew coffee with it.

  • 429 Too Many Requests

    Numerical Code: 429

    Purpose: The user has sent too many requests in a given amount of time ("rate limiting").

    Situation: When a user exceeds the allowed number of API requests per minute.

  • 451 Unavailable For Legal Reasons

    Numerical Code: 451

    Purpose: The server is denying access to the resource as a consequence of a legal demand.

    Situation: A website is blocked due to a court order.

  • 503 Service Unavailable

    Numerical Code: 503

    Purpose: The server is not ready to handle the request. Common causes are a server that is down for maintenance or that is overloaded.

    Situation: When a web server is temporarily unable to handle a request.

  • 507 Insufficient Storage

    Numerical Code: 507

    Purpose: The server is unable to store the representation needed to complete the request.

    Situation: When a server runs out of storage space.

JSON Parsing and Stringifying Example

JavaScript Code

const movieCollection = `{
    "movies": [
        {
            "title": "Pokiri",
            "director": "Puri",
            "year": 2010,
            "genre": "Crime",
            "actors": ["Mahesh Babu", "Ali"]
        },
        {
            "title": "RRR",
            "director": "Raja mouli",
            "year": 2023,
            "genre": "Sci-Fi",
            "actors": ["NTR", "Ram Charan"]
        }
    ]
}`;

const parsedData = JSON.parse(movieCollection);
console.log(parsedData);

const stringifiedData = JSON.stringify(parsedData, null, 2);
console.log(stringifiedData);
